# 🏭 Juujon 旅遊飲食部落格 - 工業風時尚設計

## 📋 設計特色

這個 CSS 設計方案專為您的 Pixnet 旅遊飲食部落格量身打造，融合**工業風**與**時尚簡約**元素。

### 核心設計理念
- **工業風格**：深灰黑色、金屬灰色調、簡潔幾何線條
- **時尚簡約**：金色點綴、大圖排版、留白設計
- **響應式**：完美適配桌面、平板、手機
- **現代美感**：流暢動畫、微互動、視覺層級清晰

## 🚀 使用方式

### 方式一：直接複製 CSS 到 Pixnet

1. 開啟 `pixnet-blog-industrial-style.css` 檔案
2. 複製全部內容
3. 登入 Pixnet 部落格
4. 進入 **設定** → **進階設定** → **自訂 CSS**
5. 貼上全部代碼
6. 儲存設定

### 方式二：使用預覽文件本地測試

1. 下載 `preview.html` 檔案
2. 用瀏覽器開啟
3. 調整視窗寬度，測試響應式效果
4. 確認效果後再套用到 Pixnet

## 🔧 自訂調整建議

如要進一步自訂，可修改 CSS 變數：

```css
:root {
  --primary-dark: #1a1a1a;      /* 改變深色系 */
  --accent-gold: #d4af37;       /* 改變金色 */
  --text-main: #2c2c2c;         /* 改變主文字顏色 */
}
```

## 💡 使用提示

1. **大圖建議**：使用 16:9 比例圖片，寬度至少 600px
2. **標籤數量**：建議 3-4 個標籤，避免過多
3. **文章摘要**：200 字內較佳，過長會影響卡片美感
4. **SEO 優化**：保留原本的 Pixnet SEO 標籤和結構
5. **測試**：不同設備上測試，確保排版完美
